This Docusaurus site is configured for deployment at:

```text
https://e2e.noetl.dev
```

Build the static site with:

```bash
npm run build
```

The build runs `npm run docs:generate` first, so the playbook inventory is regenerated from current fixture YAML files.

The generated static output is written to `build/`.
